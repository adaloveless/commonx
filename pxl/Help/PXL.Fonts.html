Unit PXL.Fonts

DescriptionUsesClasses, Interfaces, Objects and RecordsFunctions and ProceduresTypesConstantsVariables

Description

Pre-rendered bitmap fonts with Unicode support and visual effects such as border and shadow, customized spacing between individual letter pairs, rendering using vertical color gradient, formatted text among other features.

Types

TSystemFontImage = (...);

Type of image for system font.

Values
  • Font8x8: 8x8 font that was commonly used in text mode on EGA displays with 640x350 resolution.
  • Font9x8: 9x8 font that was commonly used in text mode on VGA displays with 720x400 resolution.
TTextAlignment = (...);

Text alignment when drawing with certain functions.

Values
  • Start: Text should be aligned to the beginning (either top or left depending on context).
  • Middle: Text should be centered in the middle.
  • Final: Text should be aligned to the end (either bottom or right depending on context).
